## Task 2: Rate Limiting — 保護 DataIngest endpoint

**Files:**
- Modify: `backend/Program.cs`
- Modify: `backend/Controllers/DataIngestController.cs`

**背景：** 每台設備（SerialNumber）最多每秒 5 次 ingest。使用 .NET 7+ 內建 `AddRateLimiter`，按 SerialNumber 分桶（Sliding Window）。

- [ ] **Step 1: 在 Program.cs 加 Rate Limiter 服務**

找到 `builder.Services.AddControllers();`，在其**後面**加入：

```csharp
// ── Rate Limiting（按 SerialNumber，每 10 秒最多 20 次）────────────────────
builder.Services.AddRateLimiter(options =>
{
    options.AddSlidingWindowLimiter("ingest", limiterOptions =>
    {
        limiterOptions.PermitLimit = 20;
        limiterOptions.Window = TimeSpan.FromSeconds(10);
        limiterOptions.SegmentsPerWindow = 5;
        limiterOptions.QueueProcessingOrder = System.Threading.RateLimiting.QueueProcessingOrder.OldestFirst;
        limiterOptions.QueueLimit = 0;
    });

    options.RejectionStatusCode = 429;
});
```

- [ ] **Step 2: 在 Program.cs 啟用 middleware**

找到 `app.UseCors("IoTDashboard");`，在其**後面**加入：

```csharp
app.UseRateLimiter();
```

- [ ] **Step 3: 在 DataIngestController 套用 policy**

在 `DataIngestController.cs` 的 `Ingest` 方法上加 attribute：

```csharp
using Microsoft.AspNetCore.RateLimiting;
```

並在 `[HttpPost("ingest")]` 上方加：

```csharp

```

## Task 3: LimitsController 輸入驗證

**Files:**
- Modify: `backend/Models/Dtos.cs`
- Modify: `backend/Controllers/LimitsController.cs`

**背景：** 目前 `UpdateLimitsRequest` 無任何驗證，可以存 UCL=0、LCL=999，導致所有數值都告警。

- [ ] **Step 1: 在 Dtos.cs 給 `SensorLimitDto` 加驗證 attribute**

找到 `SensorLimitDto` class，替換為：

```csharp
public class SensorLimitDto : System.ComponentModel.DataAnnotations.IValidatableObject
{
    public int SensorId { get; set; }
    public string? SensorName { get; set; }

    [System.ComponentModel.DataAnnotations.Range(0, 9999, ErrorMessage = "UCL 必須在 0~9999 之間")]
    public double UCL { get; set; }

    [System.ComponentModel.DataAnnotations.Range(0, 9999, ErrorMessage = "LCL 必須在 0~9999 之間")]
    public double LCL { get; set; }

    [System.ComponentModel.DataAnnotations.MaxLength(10)]
    public string Unit { get; set; } = "℃";

    public IEnumerable<System.ComponentModel.DataAnnotations.ValidationResult> Validate(
        System.ComponentModel.DataAnnotations.ValidationContext validationContext)
    {
        if (UCL < LCL)
            yield return new System.ComponentModel.DataAnnotations.ValidationResult(
                "UCL 必須大於或等於 LCL",
                [nameof(UCL), nameof(LCL)]);
    }
}
```

讀 `backend/Controllers/LimitsController.cs`，確認 class 上有 `[ApiController]` attribute。有的話，ModelState 驗證自動生效，**不需額外修改 Controller**。
